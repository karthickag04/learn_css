What is External CSS?

External CSS is written in separate .css files and linked to HTML documents using the <link> tag. This approach separates content from presentation, making code more maintainable and reusable.

The main advantage is that one stylesheet can control the appearance of multiple web pages, ensuring consistency across your entire website.
