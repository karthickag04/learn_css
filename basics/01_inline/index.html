This page demonstrates inline CSS. Notice how styles are written directly in HTML elements using the "style" attribute.

What is Inline CSS?

Inline CSS is applied directly to HTML elements using the style attribute. It has the highest specificity but is not recommended for large projects due to maintainability issues.

✅ Advantages

Quick to apply, highest priority, and perfect for testing styles or making small adjustments.

❌ Disadvantages

Hard to maintain, repetitive code, and mixes content with presentation.
